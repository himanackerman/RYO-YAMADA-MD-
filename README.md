## ✨ Fitur Utama

- 🤖 AI Chat & Tools
- 📥 Downloader (TikTok, YouTube, Instagram, Facebook, dll)
- 🎮 Game & RPG System
- 👥 Manajemen Grup (Anti Link, Welcome, dll)
- 🖼️ Sticker Maker + Watermark
- 📊 Sistem Level, XP, Limit
- ⚙️ Utility Tools
- 👑 Owner Tools
